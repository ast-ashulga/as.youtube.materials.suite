## Summary

The author ran a three-month personal test of ChatGPT (45/100), Windsurf (62/100), Cursor (68/100), Claude Code (74/100), and Base44 (92/100) across five criteria: speed to production, learning curve, output quality, flexibility, and value for money. ChatGPT scores lowest because it produces disconnected code snippets without project structure, deployment, or environment setup. Windsurf and Cursor genuinely speed up experienced developers but require a technical mindset and separate hosting decisions. Claude Code stands out for its autonomous build-test-debug loop, which cuts manual cleanup compared to other IDE-based tools. Base44 works differently from the rest: a single prompt produces a live, authenticated, database-backed application in minutes. The broader shift identified: AI tools are moving from "code assistant" to "outcome builder."

## Key Takeaways

- ChatGPT scored lowest (45/100) — produces disconnected code snippets without project structure, deployment, or environment setup
