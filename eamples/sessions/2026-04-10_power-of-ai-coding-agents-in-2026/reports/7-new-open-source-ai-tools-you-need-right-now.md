## Summary

Fireship covers 7 open-source AI tools for the 2026 agent stack. Agency provides agent role templates (frontend dev, backend dev, security engineer) that can be combined inside Claude Code. PromptFoo — acquired by OpenAI — is a unit-testing framework for prompts with automated red-teaming to detect prompt injection vulnerabilities. Mirrorfish is a multi-agent prediction engine that simulates a miniature AI social network fed with real-world data. Impeccable is a front-end design CLI with 17 commands (distill, colorize, animate, delight) for improving vibe-coded UIs. Open Viking is an agent-native database organizing memory, resources, and skills with tiered loading to cut token consumption. Heretic removes model guardrails from safety-tuned models like Google Gemma without post-training via its "obliteration" technique. NanoChat enables training a custom LLM for roughly $100 in GPU compute time.

## Key Takeaways

- Agency is a free open-source library of agent role templates (frontend dev, backend dev, security engineer) that can be combined inside Claude Code to add specialized domain behavior
- PromptFoo (acquired by OpenAI) acts as a unit-testing framework for prompts — compare models and run automated red-team attacks to detect prompt injection vulnerabilities
- Mirrorfish is a multi-agent prediction engine ingesting real-world data and simulating a miniature AI social network for trend forecasting
- Impeccable is an open-source front-end design CLI with 17 commands to improve the visual quality of vibe-coded UIs
- Open Viking organizes agent memory, resources, and skills with tiered loading — reducing token consumption and strengthening long-term memory
- Heretic removes model guardrails via "obliteration" from safety-tuned models like Google Gemma without expensive post-training
- NanoChat trains a custom LLM for approximately $100 in GPU compute — covering tokenization, pre-training, fine-tuning, evaluation, and a web UI

## Key Terminology

| Term | Definition |
|------|-----------|
| Slop Pipeline | AI-driven development workflow where agents auto-generate large amounts of code requiring human cleanup |
| Obliteration | Heretic's technique for removing safety guardrails from a pre-trained model without additional post-training |
| Tiered Loading | Open Viking's memory architecture loading context in priority layers to reduce token consumption |
| Red-team Attack | Automated adversarial testing to discover exploitable weaknesses like prompt injection in AI-powered features |
| Small Language Model (SLM) | Lightweight LLM trained on limited compute for affordability, speed, and full owner control |
